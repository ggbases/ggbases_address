## 如何訪問主域名
在 `C:\Windows\System32\drivers\etc\hosts` 文件中加入下面兩行：

>104.21.3.205	`ggbases.com`  
>104.21.3.205 `www.ggbases.com`  

這個 IP 一般不會變，如果失效，可以到 `https://check-host.net/check-ping?host=ggbases.com` 查詢後重新設置。
